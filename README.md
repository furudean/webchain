# webchain

webchain is a distributed [webring](https://en.wikipedia.org/wiki/Webring) webring where each member may nominate new ones, creating a walkable graph of trust.

This repository contains code for a [crawler](crawler) that can discover the webchain
and output it in different formats, and a [visualizer webpage](web) to inspect the
current state of a webchain.

See [spec.md](doc/spec.md) for the specification.
